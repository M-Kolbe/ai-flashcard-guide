Create flashcards following these principles:

ATOMICITY:
- 1 card = 1 concept (no multi-part questions!)
- Maximum 1-2 sentences per question

QUESTION FORMAT:
- Clear verbs: "Explain", "Name", "Define", "What is..."
- Include context - no open questions like "Tell me about..."
- Good example: "What is the chemical formula for water?"
- Bad example: "Tell me about water"

- Core statement in 1-2 sentences
- For formulas: explain meaning of each symbol
- For terms: definition + short example
- No technical terms without explanation

SPECIAL CASES:
- Definitions: term + definition + everyday example
- Formulas: formula + symbol explanation + numerical example
- Processes: steps in order ("Step 1: ... ‚Üí Step 2: ...")
- Comparisons: "Difference between A and B?" ‚Üí "A: [X], B: [Y]"
- Dates: date + brief relevance

- Then application cards (comparisons, processes)
